第九节 ： css基础语法讲解

<1> css语法；

CSS 层叠样式表 (Cascading Style Sheets)

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明

选择器：

通常是您需要改变样式的HTML元素

h1{width:300px;height:100px;color:red; font-size:14px;}



<2> css 值的不同写法和单位；

十六进制颜色值：

p { color: #ff0000; }

为了节约字节，我们可以使用 CSS 的缩写形式

p { color: #f00; }

六位颜色值，双数合并，即每两位合并

英文单词

p{color:red;}

<3> 空格和大小写;

	body {
	  color:  #000;
	  background: #fff;
	  margin: 0;
	  padding: 0;
	  font-family: 微软雅黑, Palatino, serif;
	}

属性和值中间可以用空格分开，但属性和值内部不可以用空格分开

css样式内部不区分大小写，可以随意使用，建议使用统一风格

<4> 选择器的分组;

h1,h2,h3,h4,h5,h6{color : green}

对选择器进行分组，用逗号将需要分组的选择器分开

同一类样式的不同标签元素可以使用分组，节省代码量

<5> CSS注释语法；

CSS注释的开始使用/*,结束使用*/

/* 注释内容 */

/* 文章标题注释 */

h1{color:red;font-size:12px;}

<6> CSS 类选择器

在 CSS 中，类选择器以一个点号显示

.center {text-align: center}

.str{color:red;font-size:12px;}

具体事例：

h1 和 p 元素都有 center 类。这意味着两者都将遵守 ".center" 选择器中的规则

<h1 class="center">文章的标题</h1>
	<p class="center">这是段落的标签</p>

<7> id 选择器

id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式

id 选择器以 "#" 来定义

具体事例：

两个 id 选择器

<p id="red">这个段落是红色。</p>
	<p id="green">这个段落是绿色。</p>

注意： id 属性只能在每个 HTML 文档中出现一次。

id 选择器和派生选择器

在现代布局中，id 选择器常常用于建立派生选择器

具体事例：

h2 strong { color: blue;}

只有在h2标签元素内部的strong标签才会改变颜色样式

<8>内嵌样式

<!--内嵌样式-->
<style type="text/css">
	.top{width:985px;height:260px;background:#FFCC66;}
	.center{width:985px;height:1000px;background:#CCCC33;}
	.center_lianxi{width:985px;height:50px;background:#FF6666;}
	.center_left{width:700px;height:950px;background:green;float:left;}
	.center_right{width:280px;height:950px;background:#CC99FF;}
	.bottom{width:985px;height:100px;background:red;}
</style>

<9>外部引入到样式

从外部引入到样式分为两种:（注意写在head标签里面）

Link样式表式：  <link rel=”stylesheet” type=”text/css” href=”my.css”(href表示路径)>   
Html式：  <style type="text/css">@import url("css.css");></style>

<10> 行内样式

<!--以style="css样式"-->
<div style="color:red;font-size:12px;"></div>
<div style="color:red;font-size:12px;"></div>
