第十节：CSS样式讲解（上）

background背景属性

background 是简写属性，在一个声明中设置所有的背景属性；

<1> background-color: 规定要使用的背景颜色;

<2> background-position: 规定背景图像的位置;

<3> background-size:规定背景图片的尺寸;

<4> background-repeat:规定如何重复背景图像;

<5> background-image:把图像设置为背景;

background-color背景颜色


.test_background{ background-color:#FC0; }

background-image背景图片


.test_background{ background-image:url("/demo/html+css/ten/images/bg.jpg"); }


背景图像的缩写形式


background:url("/demo/html+css/ten/images/bg.jpg");

background-repeat



/*不平铺*/

body{background:url("/demo/html+css/ten/images/1.png") no-repeat;}

/*平铺X*/

body{background:url("/demo/html+css/ten/images/1.png") repeat-x;}

/*平铺Y*/

body{background:url("/demo/html+css/ten/images/1.png") repeat-Y;}


是否平铺的简写形式

background:url("/demo/html+css/ten/images/1.png") no-repeat;

background-position背景图像的位置


第一个参数为垂直方向，第二个参数为水平方向;自定义百分百和像素方式，第一个参数为水平方向，第二个参数为垂直方向。

background-position:center right;

background-position:20%，50%

background-position:200px,500px;
