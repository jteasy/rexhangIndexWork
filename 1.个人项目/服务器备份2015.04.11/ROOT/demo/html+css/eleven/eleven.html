第十一节：css样式(下)

CSS文本

CSS 文本属性定义文本的外观：

color

<1>文本的颜色（color ，背景） ;

letter-spacing

<2>字符间距 （letter-spacing: 20px; letter-spacing: -0.5em;）;

line-height

<3>行间距 （line-height: 30px; line-height: 200%; line-height: 2;）;

text-align

<4>对齐文本 （text-align: center/left/right） ;

注：10px=1em, 将原来的px数值除以10，然后换上em作为单位就ok

text-decoration

装饰文本

underline{text-decoration:underline;}
.overline{text-decoration:overline;}
.line-through{text-decoration:line-through;}
.blink{text-decoration:blink;}
.inherit{text-decoration:inherit;}

text-indent

对文本进行缩进（将段落的第一行缩进）

.text-indent{text-indent:20px;}

text-transform

控制文本的大小写

.capitalize{text-transform:capitalize;}
.uppercase{text-transform:uppercase;}
.lowercase{text-transform:lowercase;}

字体属性定义文本

font-size

文本的大小

.font-size{font-size:16px;}

font-family

字体系列

.font-family{font-family:"微软雅黑";}

font-style

字体风格

.font-style{font-style:normal / italic / oblique;}

font-weight

设置文本的粗细

/*默认加粗 bold == 700*/
.bold{font-weight:bold;}
/*更粗*/
.bolder{font-weight:bolder;}
/*更细*/
.lighter{font-weight:lighter;}
/*自定义大小*/
.auto{font-weight:900;}

CSS表格

CSS 表格样式：

<1> 表格边框（border） ;

<2> 折叠边框（border-collapse:collapse;）;

<3> 表格宽度和高度 （width ，height）;

<4> 表格文本对齐（text-align）;

<5> 表格内边距 （padding）;

<6> 表格颜色 ;
